File description
1. dup.pl : code for first part for given list has duplicate element
2. sqrt.pl : code for calculating square root using binary search

----------------------------------------------------------------------
To run the code navigate to the directory and run the command
1. swipl 
2. [filename]. (to load the code)
After this give the input (similar to given below as example)

----------------------------------------------------------------------
Example input and corresponding output 
----------------------------------------------------------------------
Part 1

?- [dup].
true.

?- dup([1,2,3,4,5,a,c,3,a]).
true .

-----------------------------------------------------------------------
Part 2 

?- [sqrt].
true.

?- sqrt(25,R,0.0000001).
R = 4.999999996895592 .
